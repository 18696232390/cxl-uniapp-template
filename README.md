# cxl-uniapp-template
💡自用的uniapp的快速开发脚手架

### 脚手架说明
1.集成uview-ui框架 ： https://uviewui.com/ 💡
2.集成mypUI框架 ： https://github.com/wakaryry/mypUI 💡

> 大部分是套用的UI组件，对于脚手架的易用性后面慢慢完善
